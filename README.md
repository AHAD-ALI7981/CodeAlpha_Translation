# 🌍 CodeAlpha - Language Translation Tool

This project is developed as part of the CodeAlpha AI Internship.

## 📌 Project Description
This is a simple Language Translation Web App built using Streamlit and Deep Translator API.  
Users can enter text, select source and target languages, and get translated output instantly.

## 🛠 Technologies Used
- Python
- Streamlit
- Deep Translator

## 🚀 How to Run the Project

1. Clone the repository:
```
git clone https://github.com/AHAD-ALI7981/CodeAlpha-Translation.git
```

2. Navigate to the project folder:
```
cd CodeAlpha-Translation
```

3. Install dependencies:
```
pip install -r requirements.txt
```

4. Run the application:
```
streamlit run translation.py
```

## 📷 Features
- Multiple language support
- Easy-to-use interface
- Fast translation response
